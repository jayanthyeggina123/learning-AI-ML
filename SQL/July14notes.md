# Temporary Table
A **Temporary Table** is a table that exists only for the current database session or connection. It is used to store temporary data and is automatically deleted when the session ends.

---

# Recursive CTE (Common Table Expression)
A **Recursive CTE** is a Common Table Expression that repeatedly executes itself **until a specified stopping condition is satisfied**. It is commonly used to work with hierarchical or recursive data.

---

# View
A **View** is a **virtual table** created from one or more tables. It stores only the SQL query (not the actual data), making it a **blueprint to view data**.

---

# Stored Procedure
A **Stored Procedure** is a saved SQL program that contains one or more SQL statements. It works similarly to a function in programming and can be executed whenever needed.

---

# DELIMITER
A **Delimiter** is a character or symbol that tells MySQL where one SQL statement ends. The default delimiter is a semicolon (`;`).

---

# Dynamic Stored Procedure
A **Dynamic Stored Procedure** is a stored procedure that builds and executes SQL statements at runtime, allowing queries to change dynamically based on input or conditions.

---

# Information Schema
**Information Schema** is a system database that stores **metadata** (data about the database). It contains information about databases, tables, columns, views, indexes, constraints, and other database objects.
